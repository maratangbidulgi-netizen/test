# test# 🎮 Vanilla JS Tetris

HTML5 Canvas와 순수 JavaScript로 구현한 테트리스 게임입니다.  
별도의 라이브러리 없이 **게임 로직, 렌더링, 입력 처리**를 모두 직접 구현했습니다.

---

## 📌 주요 기능

- ⬛ **정통 테트리스 게임 플레이**
- 🔳 **전체 플레이 화면 그리드 오버레이**
- 🎨 **그라디언트 블록 텍스처**
- 🧮 **점수 시스템**
  - 한 줄 삭제 시 **10점**
  - 점수 10점 증가마다 낙하 속도 증가
  - **100점 도달 시 최고 속도 고정**
- 🔄 **R 키로 즉시 재시작**
- ⚡ **하드 드롭 / 소프트 드롭**
- 🧱 **벽 충돌 보정 회전 (Wall Kick)**
- 💥 **게임 오버 처리**

---

## 🎮 조작 방법

| 키 | 동작 |
|----|------|
| ← / → | 좌우 이동 |
| ↓ | 소프트 드롭 |
| ↑ | 회전 |
| Space | 하드 드롭 |
| R | 게임 재시작 |

---
